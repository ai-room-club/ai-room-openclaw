# OpenClaw RU Installer

> **Что это:** публичный установщик OpenClaw-агента на VPS для русскоязычных пользователей. Ты скармливаешь этот репозиторий Claude Code или Codex CLI, и AI-агент сам выполняет установку у тебя на сервере.
>
> **Для кого:** абсолютный новичок. Не требует знания терминала, SSH или настройки серверов.
>
> **Сколько времени:** ~60 минут.
>
> **Сколько денег:** от ~200–400 ₽/мес (любой VPS с Ubuntu 22.04+) + твоя действующая подписка ChatGPT Plus/Pro или Claude Pro/Max.

## 🎯 Что ты получишь

По итогам установки у тебя будет:

- **Telegram-бот**, которому ты пишешь в любое время.
- **AI-агент**, который живёт на твоём VPS 24/7, отвечает через бот.
- **Память** между разговорами (не забывает, о чём вы общались вчера).
- **Умение действовать** — искать в вебе, проверять почту, работать с файлами и сервисами (скилы добавляются отдельно).
- **Базовая безопасность** — бот отвечает только тебе.

### Короткая версия

1. Купи любой VPS с **Ubuntu 22.04+** или **Debian 12+** (минимум 2 GB RAM, 2 vCPU, 20 GB диск). Подойдёт любой провайдер — главное, чтобы ОС подходила.
2. Установи Claude Code или Codex CLI (см. ссылки выше).
3. Клонируй этот репо:
   ```bash
   git clone https://github.com/ai-room-club/ai-room-openclaw.git
   cd ai-room-openclaw
   ```
4. Открой в VS Code (для Claude Code) или запусти `codex` в терминале.
5. Скопируй промпт из соответствующего `PROMPT-*.md` и вставь в чат.
6. AI начнёт диалог — отвечай на его вопросы, давай доступ к серверу.
7. Через ~60 минут у тебя работающий Telegram-бот с AI-мозгами.

## ⚠️ Что нужно заранее

| Обязательно | Зачем |

| **Банковская карта или СБП** | Оплата VPS |
| **~60 минут свободного времени** | Чтобы не прерываться |

## 🛡️ Что AI делает на твоём сервере

**Полный список действий описан в [`RUNBOOK.md`](./RUNBOOK.md).** Коротко:

✅ **AI делает:**

- Подключается к твоему VPS по SSH (только по IP, который ты указал).
- Создаёт non-root пользователя, настраивает SSH-ключи, включает firewall (UFW), устанавливает fail2ban.
- Устанавливает OpenClaw через официальный `install.sh`.
- Настраивает systemd-службу (чтобы агент работал 24/7).
- Подключает твою ChatGPT-подписку как «мозг» (через OAuth).
- Создаёт конфигурацию бота в Telegram (токен ты получаешь сам у @BotFather).
- Настраивает allowlist — бот отвечает только тебе.
- Запускает security audit, исправляет critical findings.

❌ **AI НЕ делает:**

- Не трогает другие сервера, кроме указанного тобой.
- Не устанавливает ничего на твой локальный компьютер.
- Не импровизирует команды — использует только проверенные скрипты из `scripts/`.
- Не продолжает автоматически при ошибках — всегда спрашивает тебя.
- Не передаёт твои данные куда-либо, кроме твоего VPS.

## 🆘 Если что-то пошло не так

1. **Ошибка в процессе установки** — AI сам диагностирует и предложит решение (в RUNBOOK есть «Failure recovery» для каждой фазы).
2. **Хочешь остановить AI** — напиши в чате «Остановись. Не делай ничего больше».
3. **Не понимаешь, что AI пытается сделать** — напиши «Объясни простыми словами, что ты сейчас хочешь сделать и зачем».
4. **AI явно галлюцинирует** (команды не из scripts/) — прерывай и пиши в [issues](https://github.com/ai-room-club/ai-room-openclaw/issues).
5. **Полный гайд на русском** (альтернатива ручному пути, без AI-установщика) — **[Notion-гайд AI Room Club](https://airoomclub.ru)**.

## 🔄 После установки

**Что попробовать сразу:**

- Напиши боту «Привет, расскажи о себе».
- Спроси «Какой сегодня день недели?» (проверь, что работает базовый access к datetime).
- Задай сложный вопрос — проверь качество модели.

**Что настроить дальше (в следующий заход):**

- **SOUL.md** — дай агенту имя, характер, специализацию. Шаблоны — в [`templates/soul/`](./templates/soul/).
- **Skills** — подключи Gmail, Google Calendar, веб-поиск. Официальный каталог: [clawhub.ai](https://clawhub.ai).
- **Dashboard через SSH-туннель** — управление через веб-интерфейс.
