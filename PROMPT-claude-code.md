# Промпт для Claude Code

Скопируй этот промпт целиком, открой Claude Code в VS Code и вставь.

---

## Шаг 1. Подготовка (в VS Code)

1. Открой VS Code.
2. Установи Claude Code расширение, если ещё не установил ([claude.com/claude-code](https://claude.com/claude-code)).
3. Залогинься (нужна подписка Claude Pro или Max, либо API-ключ Anthropic).
4. Клонируй репо с установщиком:
   ```
   git clone https://github.com/ai-room-club/ai-room-openclaw.git
   cd openclaw-ru-installer
   code .
   ```
5. Открой Claude Code в sidebar VS Code.

## Шаг 2. Активационный промпт (копипаста)

Скопируй следующее **целиком** в чат Claude Code:

```
Привет. Я хочу установить OpenClaw-агента на свой VPS через этот установщик.

Ты сейчас:

1. Прочитай @CLAUDE.md — там твоя роль и правила.
2. Прочитай @RUNBOOK.md — там источник истины по установке.
3. Начни Phase 0 (preflight): задай мне по одному вопросу данные, которые тебе нужны для установки.

Я абсолютный новичок — не знаю терминала, SSH и прочих технических деталей. Объясняй мне всё простым русским языком. Задавай вопросы по одному, жди ответа, прежде чем переходить к следующему.

Начинай.
```

## Что произойдёт дальше

Claude Code прочитает инструкцию и начнёт диалог с тобой. Он будет:

1. **Задавать тебе вопросы** (IP сервера, пароль, подтверждения) — по одному.
2. **Объяснять каждый шаг** простым языком.
3. **Запускать команды на твоём сервере** — только после твоего согласия, только из готовых скриптов в `scripts/`.
4. **Отчитываться о результате** каждой фазы.

## Что тебе понадобится в процессе

- **Купленный VPS с Ubuntu 22.04+** или Debian 12+ (любой провайдер). IP и пароль root на руках.
- **Рабочий VPN** — понадобится один раз на Phase 5 (OAuth к ChatGPT).
- **Подписка ChatGPT Plus или Pro** — как «мозг» агента.
- **Активный Telegram-аккаунт** — где будет жить бот.
- **~60 минут свободного времени** — чтобы не прерываться.

## ⚠️ Что важно понимать про безопасность

Claude Code получит **SSH-доступ** к твоему серверу. Он:

✅ **Может:** подключаться к серверу, запускать команды из scripts/ этого репо, читать и писать файлы на сервере.

❌ **Не может:** трогать другие сервера, устанавливать OpenClaw на твой **локальный компьютер**, импровизировать команды вне scripts/, работать без твоего подтверждения на destructive-действиях.

Все его действия описаны в `CLAUDE.md` и `RUNBOOK.md`. Если увидишь, что Claude Code пытается делать что-то **не по RUNBOOK** — остановись и напиши «Остановись и объясни, зачем тебе это».

## Если что-то пойдёт не так

- **Claude Code завис** или отвечает странно → напиши «Покажи текущую фазу и статус». Он восстановит контекст.
- **Ошибка на сервере** → Claude Code сам обратится к «Failure recovery» соответствующей фазы RUNBOOK.
- **Не хочешь продолжать** → напиши «Остановись. Не делай ничего больше». Claude Code прервёт процесс.
- **Нужен fallback** (без AI) → см. [Notion-гайд AI Room Club](https://airoomclub.ru).

## Альтернатива — Codex CLI

Если у тебя нет Claude Pro/Max, но есть ChatGPT Plus/Pro — можешь использовать **Codex CLI** вместо Claude Code. Промпт почти идентичный, см. [`PROMPT-codex.md`](./PROMPT-codex.md).
